## Getting Started

### STEP 1: Clone the project

```bash
git clone https://github.com/ScaleupFinance/currency-calculator.git
```

### STEP 2: Open the folder

```bash
cd currency-calculator
```

### STEP 3: Install dependecies

```bash
npm install
# or
yarn
```

### STEP 4: Start the development server

```bash
npm run dev
# or
yarn dev
```
